# SFTP Uploader
Upload files to server over SFTP

# Usage

## Change variables
```
ip = '<HOST>' (without < and >)
port = '<PORT>' (without < and >)
username = '<USERNAME>' (without < and >)
password = '<PASSWORD>' (without < and >)
```

## Run python file
```
python3 run.py
```
